LonWord

Holds a 16-bit numerical value.

The LonWord structure holds a 16-bit unsigned value in big-endian ordering through two separate high-order and low-order bytes.  Use the <LON_SET_SIGNED_WORD> or <LON_SET_UNSIGNED_WORD> macro to obtain the signed or unsigned numerical value in the correct byte ordering.
